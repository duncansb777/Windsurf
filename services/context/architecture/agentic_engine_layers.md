# DXC Agentic Engine Layers

- **Experience**: Console UI, Health Agents screen, Studio, Eval dashboard
- **Orchestration**: DeepAgent graphs, task routing, autonomy controls
- **Skills (MCP)**: CCS tools, EHR tools, directory search, consent checks
- **Reasoning**: Plan/explain/predict pipelines, safety filters
- **Data**: Connectors to CCS, EPIC, HCA; caching and audit
- **Governance**: SACSF/SAPPP controls, decision boundaries, logging
- **Ops**: Telemetry, AgentOps instrumentation, rollouts, canaries
