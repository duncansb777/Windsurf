# SACSF Controls (Excerpt)

- **SACSF-AC-1 Access Control**
  Enforce least privilege; authenticate and authorize all tool calls.

- **SACSF-AC-3 Purpose of Use**
  Record and enforce purpose-of-use for every data retrieval/write.

- **SACSF-LG-1 Logging & Audit**
  Log inputs/outputs and decisions; retain for audit; protect PII.

- **SACSF-DP-2 Data Protection**
  Apply minimization and masking for sensitive fields.

- **SACSF-CR-1 Change Control**
  Validate changes against constraints; require tests and approvals.
